S O M M A I R E

Agent et candidature

Cette table permet de saisir les parametres de base :
- Nom
- Prenom
- adresse
- cp
- ville
- service
- grade
Le cp et la ville sont définis par defaut dans dyn/var.inc

Dans l onglet candidature, il est saisi la candidature :
- pour un scrutin
- dans un bureau
- pour une periode : matin, apres midi ou journee
- pour un poste : planton, secretaire

La decision permet de prendre en compte les candidatures et ensuite, il faut saisir les heures effectuées pour le paiement des heures supplementaires ( requete : scrutin-heure associé sous ooo/scrutinheure.ods)
