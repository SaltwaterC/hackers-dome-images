Elu et affectation

Cette table permet de saisir :
- Nom
- Prenom
- Nomjf
- date naissance
- lieu de naissance
- adresse
- cp
- ville
- candidat pour delegue et assesseur

Le cp et la ville sont définis par defaut dans dyn/var.inc

Dans l onglet affectation, il est saisi l'affectation :
- pour une scrutin
- dans un bureau
- pour une periode : matin, apres midi ou journee
- pour un poste : president, vice president, assesseur, delegue
