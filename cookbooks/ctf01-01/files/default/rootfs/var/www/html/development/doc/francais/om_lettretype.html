vous devez avoir les droits d ecriture dans le repertoire sql ...

Paramétrage des lettres type

Pour parametrer les états il faut paramétrer:
- Parametres généraux
- Parametres du titre
- Parametres du corps
- Requete SQL
Parametrage généraux

- orientation: portrait ou paysage
- Format : A4, A3
- logo: logo défini en collectivite
* positionnement left/top du logo


Parametrage du titre

- titre : texte du titre
* variable: £idx, £aujourdhui ... définies en dyn/varlettretypepdf.inc
* entre [] zones définies en requete SQL : exemple [nom]
- positionnement left/top du titre
- largeur du titre : / 0 = largeur automatique
- hauteur de chaque ligne(suivant taille de la police)
- Police (font), attribut, taille, bordure, alignement du titre

Parametrage du corps

- corps : texte du corps
* variable: £idx, £aujourdhui ... définies en dyn/varlettretypepdf.inc
* entre [] zones définies en requete SQL : exemple [nom]
- positionnement left/top du corps
- largeur du titre : / 0 = largeur automatique
- hauteur de chaque ligne (suivant taille de la police)
- Police (font), attribut, taille, bordure, alignement du corps

Requete SQL

- sql : texte de la requete
* variable: £idx, £aujourdhui ... définies en dyn/varlettretypepdf.inc
* entre [] zones définies en requete SQL : exemple [nom]
