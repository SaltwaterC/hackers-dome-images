vous devez avoir les droits d ecriture dans le repertoire sql ...

Paramétrage des SOUS ETATS

Pour parametrer les états il faut paramétrer:
- Parametres du titre
- Entete du tableau
- Data du tableau
- Cellule du tableau
- Total du tableau
- Moyenne du tableau
- nombre d enregistrement du tableau
- operations du tableau
- sql (la requete sql de l etat)

Titre

- titre : texte du titre
* variable: £idx, £aujourdhui ... définies en dyn/varetatpdf.inc
* entre [] zones définies en requete SQL : exemple [nom]
- hauteur de chaque ligne(suivant taille de la police)
- Police (font), attribut, taille, bordure, alignement du titre

- fond du titre
* avec ou sans (transparent)
* couleur du fond (avec fond)=> choix couleur:
- couleur du texte => choix couleur:
- intervalle début (avant tableau) , fin (après tableau)

Entete du tableau

il s agit de l entete de colonne du tableau qui définissent les zones de la requete sql (ex select nom_usage as nom from ...) affichage de nom
 - avec ou sans entete
- fond de l entete
* avec ou sans (transparent)
* couleur du fond (avec fond)=> choix couleur:
- couleur du texte => choix couleur:
- hauteur de l entete

bordure et alignement pour chaque colone SQL
- bordure pour chaque colonne: T = top, L= left, B = bottom, R= right
- alignement de chaque colone : C= center, L= left, R =right, J = justify

Data du tableau

il s agit des données du tableau définies dans les zones de la requete sql

- largeur du tableau (paramétre général)
- bordure
* avec ou sans (transparent)

* choix couleur de la bordure:

pour une meilleure lecture des données il est possible d alterner 2 couleurs de fond

- couleur du fond1 => choix couleur:
- couleur du fond2 => choix couleur:

Cellule du tableau

il s agit des cellules contenant les données définies dans les zones de la requete sql

- fond de la cellule
* avec ou sans (transparent)
- hauteur de la cellule
par défaut, la police de caractere est reduite si le texte dépasse et le retour charriot est effectué pour les zones textes

- largeur de chaque colone: le total des largeurs de colones doit etre égal à la largeur du tableau

bordure et alignement pour chaque colone SQL

- bordure pour chaque colonne: T = top, L= left, B = bottom, R= right
- bordure1 pour chaque colonne: T = top, L= left, B = bottom, R= right
- alignement de chaque colone : C= center, L= left, R =right, J = justify

Operations du tableau


Numerique, total, moyenne, nombre pour chaque colone SQL

- numerique permet un affichage des numerique: 999: zone alphanumerique, 0 (nombre entier), 1 (1 decimale), 2 ( 2 decimales) ...

Ces zones sont à parametrer si il est souhaité une operation (total, moyenne, nombre)

- total : 0 colone sans, 1 colone avec
- moyenne : 0 colone sans, 1 colone avec
- nombre : 0 colone sans, 1 colone avec
