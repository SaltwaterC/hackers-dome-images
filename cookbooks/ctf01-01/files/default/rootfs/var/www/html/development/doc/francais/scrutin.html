S O M M A I R E

Scrutin / selection des candidats et des elus

Vous pouvez faire:
- lettre type convocation agent dont les candidatures sont retenues
- lettre type convocation des presidents
- lettre type recepisse membre du bureau
- etat president / vice president par bureau
- etat par bureau

Le formulaire scrutin permet de saisir :
- libelle du scrutin
- date
- tour
- date naissance
- canton concerne ou toute la commune (controle des candidatures et affectation)
- texte sur la convocation agent
- texte sur la convocation des presidents
Le cp et la ville sont définis par defaut dans dyn/var.inc

Deux onglets :
- agent pour le scrutin
- elu pour le scrutin
